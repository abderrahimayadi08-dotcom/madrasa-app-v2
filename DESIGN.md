# Design System

## Theme
- Material 3
- Green seed (#1B5E20) — Islamic/traditional feel
- Light mode default, dark mode available
- RTL support (Arabic-first)

## Color Palette
- Primary: Green (#1B5E20 → #4CAF50)
- Secondary: Teal (#00796B)
- Error: Red (#D32F2F)
- Surface: White / Near-white
- Background: Warm off-white (#FAFAF5)

## Status Colors
- Pending: Orange (#FF9800)
- Approved: Green (#4CAF50)
- Rejected: Red (#F44336)
- Hold: Grey (#9E9E9E)
- Urgent: Deep red (#D32F2F)
- Medium: Orange (#FF9800)
- Low: Green (#4CAF50)

## Typography
- Arabic-first with Material 3 default (Cairo or Noto Naskh Arabic if available)
- Large headings for screen titles
- Body text at 14-16sp for readability
- Small text at 12sp for metadata

## Spacing
- 4dp base unit
- 16dp default padding (screens)
- 12dp card horizontal padding
- 8dp between form fields
- 24dp between sections

## Shape
- Small: 8dp radius (cards, buttons)
- Medium: 12dp radius (dialogs, bottom sheets)
- Large: 16dp radius (images)

## Components
- Cards: slight elevation, 8dp radius, no nested cards
- Buttons: FilledButton primary, OutlinedButton secondary
- Chips: FilterChip for status/priority selection
- AppBar: centered title, no elevation
- FAB: circular, primary color
